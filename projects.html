SpySee

A Reconnaissance robot built for Military.

  • Next

Introduction

Modern Warfare is greatly fuelled by the accuracy and efficiency of autonomous robots and targeting systems. SpySee was developed with a vision to strengthen tactical capabilities of military through integrating Artificial Intelligence tools into Robotics.

This robot will make it easier for our armed forces to spy on enemy bases more autonomously and safely. It uses computer vision and deep learning algorithms with a rigid framework to process the data from various sensors so that reconnaissance automation can be achieved.

Features

  • Live Video Feed
  • Object Detection
  • Face Recognition
  • Easy Android Interface
  • Automatic Geo-tagging

Integration

The robot streams the video feed from camera to SpySee Android app, through which all the functions of robot are controlled. The robot is based on Raspberry pi 3b which handles all the communication, control and computation of the data received from the app and applies deep learning algorithms to recognise the objects, faces, voice and its path. The physical appearance of the robot will make it hard for the enemy to see or capture the robot achieved through camouflage techniques, it also features a self-destruct mode in case of being captured.Robot's design isbased reliable tank design so that it can move effectively and efficiently in any terrain. Features like person follower, enemy base mapping, heat sensors etc. can be integrated in future.

CAD

SpySee is designed on Fusion 360 by Autodesk. Design is optimised for Sheet Metal work with minimum number of pieces for faster and cheaper prototyping. All tests and framing of concepts into prototype were performed smoothly on Fusion 360.
